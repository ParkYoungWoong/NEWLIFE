<a></a>

anchor 태그는 특정 문서를 링크(link)하는 기능을 하며 href(Hyper text REFerence) 속성 등을 이용하여 특정 문서의 경로를 설정

속성 (Attributes)

href
링크 문서의 경로(URL)를 입력하세요. 
<a href="http://www.google.com">Google</a>
별도의 경로가 준비되지 않았다면 #을 입력하세요. 
<a href="#">Google</a>
href속성을 사용하지 않으면 하나의 빈 링크를 생성합니다. 
<a>Google</a>
target
링크 문서의 내용을 표시할 위치(창, 탭) 설정.
_self
링크된 내용이 현재의 창(탭)에서 보여집니다. 
<a href="#" target="_self">Google</a>
_self는 기본값이기 때문에 생략이 가능합니다. 
<a href="#">Google</a>
_blank
링크된 내용이 새로운 창(탭)이 열리면서 보여집니다. 
<a href="#" target="_blank">Google</a>
type
링크 문서의 타입
text/html
HTML 문서를 링크했을 때
text/css
CSS 문서를 링크했을 때
text/javascript
CSS 문서를 링크했을 때
기타 문서 타입
hreflang
링크 문서의 언어를 지정
en
영어 (English)
ko
한국어 (Korean)
rel
현재 문서와 링크 문서의 관계를 지정
media
링크 문서에 최적화된 미디어/디바이스를 지정(참고용)
